# 🎨 Hirst Dot Painting Generator

A Python Turtle Graphics project that generates a colorful dot painting inspired by the artwork of Damien Hirst.

## Features

- Generates a grid of randomly colored dots.
- Uses RGB color values extracted from an image.
- Inspired by Damien Hirst's famous spot paintings.
- Built using Python's Turtle Graphics module.

## Preview

The program creates a 10 × 10 grid of colored dots with randomly selected colors from a predefined palette.

## Technologies Used

- Python 3
- Turtle Graphics
- Random Module
- Colorgram (optional, for color extraction)
