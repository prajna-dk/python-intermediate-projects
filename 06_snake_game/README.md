# 🐍 Snake Game

A classic Snake Game built using Python's **Turtle Graphics** module and Object-Oriented Programming principles.

The player controls the snake using the arrow keys, collects food to grow longer, and avoids colliding with the walls or its own body. The score increases with every piece of food collected, and the game ends upon collision.

---

## 📌 Features

- Classic Snake gameplay
- Smooth snake movement
- Random food generation
- Real-time score tracking
- Snake grows after collecting food
- Collision detection with walls
- Collision detection with the snake's own body
- Keyboard controls using arrow keys
- Modular Object-Oriented design

## ▶️ How to Run

1. Clone the repository.

2. Navigate to the project folder.

3. Run:

```bash
python main.py
```

4. Control the snake using the keyboard:

- ⬆️ Up Arrow
- ⬇️ Down Arrow
- ⬅️ Left Arrow
- ➡️ Right Arrow

---

## 🎮 Gameplay

- Guide the snake to collect food.
- Each food collected increases your score.
- The snake grows longer after every successful collection.
- Avoid crashing into the walls.
- Avoid colliding with your own body.
- The game ends when a collision occurs.
