# 🏓 Pong Game

A classic two-player Pong game built using Python's **Turtle Graphics** module and Object-Oriented Programming principles.

Players control paddles from opposite sides of the screen, bounce the ball back and forth, and score points when their opponent misses the ball. The game becomes progressively faster after every successful paddle hit, making gameplay increasingly challenging.

---

## 📌 Features

- Classic two-player Pong gameplay
- Smooth ball animation
- Two-player keyboard controls
- Ball collision with paddles
- Ball collision with walls
- Increasing ball speed after each paddle hit
- Automatic score tracking
- Modular Object-Oriented design

---

## 🎯 Gameplay

- Bounce the ball back using your paddle.
- Prevent the ball from passing your side.
- Score a point when your opponent misses.
- The ball speeds up after every successful paddle hit.
- The match continues until the players decide to stop.
