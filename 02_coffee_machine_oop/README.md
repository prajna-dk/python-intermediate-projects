# ☕ Coffee Machine Simulator using Object-Oriented Programming (OOP)

## Overview

This project is a command-line Coffee Machine Simulator built using Python's Object-Oriented Programming (OOP) principles.

Unlike the previous version where the application logic was implemented manually, this project utilizes pre-built classes and methods to manage menu operations, resource availability, coffee preparation, and payment processing.

The project demonstrates how multiple classes can work together to create a modular, reusable, and maintainable application.

---

## Features

* Select from multiple coffee options
* Display available drinks
* Check ingredient availability
* Process coin-based payments
* Prepare and serve coffee
* Generate machine reports
* Turn the machine on and off
* Handle invalid user input

## How It Works

1. The user selects a coffee from the menu.
2. The system retrieves the corresponding drink object.
3. The CoffeeMaker checks if enough ingredients are available.
4. The MoneyMachine processes the user's payment.
5. If payment is successful, the coffee is prepared.
6. Users can request a machine report or stop the program at any time.
