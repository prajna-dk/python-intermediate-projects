# 🧠 Python Quiz Game

An interactive command-line quiz game built using **Object-Oriented Programming (OOP)** in Python.

This application asks users a series of True/False questions, validates their answers, keeps track of their score, and displays the final result once the quiz is complete.

This project was built as part of my Python programming journey to strengthen my understanding of classes, objects, modular programming, and application design.

---

## 🚀 Features

- Interactive command-line interface
- True/False quiz questions
- Accepts both:
  - True / False
  - T / F
- Automatic answer validation
- Running score tracker
- Final score summary
- Modular project structure

## ⚙️ How It Works

1. Loads quiz questions from `data.py`.
2. Converts each question into a `Question` object.
3. Stores all questions in a question bank.
4. Creates a `QuizBrain` object to manage the quiz.
5. Displays one question at a time.
6. Validates the user's answer.
7. Updates the score.
8. Displays the final score after all questions have been answered.
